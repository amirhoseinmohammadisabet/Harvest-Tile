# 🌾 Tile Farm

A cozy, real-time web-based farming simulator. Plant crops, craft artisan goods, trade on a global market, and adapt to changing seasons in this data-driven idle game.

## ✨ Features
* **Real-Time Farming:** Crops grow based on real-world time. Upgrade to Scythes and Planters to automate your workflow.
* **The Shed (Crafting):** Unlock the Shed to process raw crops into high-value artisan goods using Kegs and Juicers.
* **Dynamic Weather & Seasons:** A full 365-day calendar with 4 seasons. Rain speeds up crop growth, while snow slows it down. Watch the visual weather effects change the farm's atmosphere!
* **Player Market:** Buy and sell goods using a live cloud database. Check your Mailbox to claim your earnings.
* **Global Inventory:** Keep track of all your raw materials, seeds, and crafted goods in one unified dashboard.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend / Database:** Firebase (Authentication & Cloud Firestore)
* **Architecture:** Data-driven design using local JSON files for items, machines, and weather states.

## 🚀 How to Run Locally
Because this game uses JavaScript Modules (ES6) for the Firebase backend, you must run it on a local server.
1. Clone the repository to your machine.
2. Open the project folder in Visual Studio Code.
3. Install the **Live Server** extension.
4. Right-click `index.html` and select **"Open with Live Server"**.

## 🎮 How to Play
1. Sign in using your Google account.
2. Buy empty lots and plant your starter Wheat seeds.
3. Harvest your crops, sell them for profit, and unlock new seeds like Strawberries and Starfruit.
4. Save up $50,000 to unlock The Shed and start brewing Artisan Goods for massive profits!
